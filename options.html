How to get an OpenAI API Key:

  1. Sign up or log in at OpenAI's platform
  2. Go to the API Keys section in your account
  3. Create a new secret key
  4. Copy and paste it here
  5. Note: Using the OpenAI API will incur costs based on your usage

How to use a fine-tuned model:

  1. Fine-tune a model on OpenAI's platform
  2. Once fine-tuning is complete, get your model's name
  3. Add the model name to the dropdown list above
