FanFix Chat Assistant Options

How to get an OpenAI API Key:

  1. Sign up or log in at OpenAI's platform
  2. Go to the API Keys section in your account
  3. Create a new secret key
  4. Copy and paste it here
  5. Note: Using the OpenAI API will incur costs based on your usage

How to use OpenAI Assistants:

  1. You can create a new Assistant using the "Create New Assistant" button above
  2. Or you can create an Assistant manually in the OpenAI Assistants interface
  3. Copy your Assistant ID (starting with "asst_") and paste it above
  4. The Assistant should be designed to suggest engaging replies for FanFix conversations
